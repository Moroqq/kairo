## Что за проект

**KAIRO** — персональный таск-менеджер с терминально-матричной эстетикой
(моноширинный шрифт, зелёное-на-чёрном по умолчанию, «матричный дождь»,
8 тем оформления). Весь UI на русском. Данные — в `localStorage`
(пока без бэкенда, каждое устройство хранит своё).

Стек: **Vite 5 + React 18 + TypeScript + Tailwind + framer-motion + react-query + zustand**.
Десктоп-обёртка: Tauri (папка `src-tauri/`).
Git author для статистики: `Moroqq <svolleen@gmail.com>`. Коммитим сразу в `main`.

### Разделы (роуты в `src/App.tsx`)
- `/` — **Фокус**: одна важная задача + действия (главный экран)
- `/board` — **Доска**: канбан (мобайл — вкладки-колонки со свайпом)
- `/calendar` — **План**: месячный календарь + дневник (`DayView`) + расписания-паттерны
- `/todo` — **Листок**: дневной to-do
- `/weeks` — **Итоги**: недельная ретроспектива
- `/expenses` — **Траты**: регулярные платежи (сумма, день месяца, дни до оплаты)
- `/log` — **События**: журнал

### Архитектурный паттерн (повторять для новых фич)
`types/*.ts` → `lib/*-storage.ts` (localStorage + миграция) →
`services/*.service.ts` (логика) → `hooks/use*.ts` (react-query) →
`pages/*Page.tsx` + `components/*`. Навигация — единый список `src/components/layout/nav.ts`.
Стилизация — CSS-переменные в `src/styles/globals.css`, темы в `src/themes/themes.ts`.
Мобайл — хук `useIsMobile()` (`src/hooks/useMediaQuery.ts`, max-width 767px).

## Как запустить

```bash
npm install
npm run dev          # https://localhost:1420  (HTTPS включён!)
npx tsc --noEmit     # проверка типов перед коммитом
```

**HTTPS обязателен** — экран входа использует `crypto.subtle` (PBKDF2),
а он работает только в защищённом контексте (https / localhost).
Плагин `@vitejs/plugin-basic-ssl` в `vite.config.ts` даёт самоподписанный серт.

**С телефона** (та же Wi-Fi): `https://<IP-ПК>:1420` → принять предупреждение
о сертификате. Если не открывается — брандмауэр Windows:
`New-NetFirewallRule -DisplayName "Vite 1420" -Direction Inbound -LocalPort 1420 -Protocol TCP -Action Allow` (PowerShell от админа).

### AI-функции (опционально, нужны ключи)
`.env` (в `.gitignore`, не коммитить):
```
VITE_PARSER_KEY=sk-ant-...     # Anthropic — парсинг текста/фото в задачу
VITE_STT_KEY=sk-proj-...       # OpenAI Whisper — распознавание голоса (рус.)
```
Голос: Whisper (речь→текст) → Claude (текст→структура задачи). Без ключей
работает только ручное создание задач.
